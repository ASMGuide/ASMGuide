What does this guide install?

This guide has the end goal of taking an ASM laptop and installs a clean version of Windows 10.
This disables all of the school's tracking programs (Omnito and Lanschool), giving you freedom.

Doing this will COMPLETELY DELETE everything currently installed on the computer! You will have to reinstall Google Chrome, Microsoft Office and other programs.
Files stored on the computer will also be deleted.

You have to be careful with the tech people, because you can get in a lot of trouble for following this guide.

If you don't want to risk getting in trouble, close this tab.

If you don't mind the warnings, you can get started.
